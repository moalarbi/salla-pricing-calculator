# حاسبة تسعير سلة - Salla Pricing Calculator

صفحة هبوط تفاعلية لحساب سعر البيع المقترح للمنتجات بعد احتساب جميع الرسوم والشحن والضرائب.

## المميزات

✅ **حساب ذكي للأسعار** - احسب سعر البيع المقترح بعد احتساب:
- رسوم بوابات الدفع (مدى، فيزا/ماستر، STC Pay، تابي، تمارا)
- الرسوم الثابتة
- ضريبة الرسوم (15% افتراضياً)
- تكاليف الشحن
- الربح المستهدف

✅ **طرق دفع مرنة**:
- اختيار طريقة دفع واحدة
- متوسط موزون (توزيع نسب مختلفة لكل طريقة)

✅ **سقف رسوم مدى** - تفعيل اختياري لتحديد الحد الأقصى لرسوم مدى

✅ **تفصيل شامل** - عرض تفصيلي لجميع الرسوم والحسابات

✅ **واجهة سهلة الاستخدام** - تصميم عربي RTL احترافي

✅ **حفظ تلقائي** - حفظ جميع الإعدادات والمدخلات في localStorage

✅ **Responsive Design** - يعمل بشكل مثالي على الهاتف والتابلت والحاسوب

## التقنيات المستخدمة

- **HTML5** - بنية الصفحة
- **CSS3** - تنسيق احترافي مع دعم RTL
- **Vanilla JavaScript** - منطق الحساب والتفاعل (بدون مكتبات خارجية)

## شرح الحسابات

### المتغيرات الأساسية

- **C** = تكلفة المنتج (COGS)
- **SH** = تكلفة الشحن
- **TP** = الربح الصافي المستهدف
- **P** = نسبة الرسوم (موزونة أو طريقة واحدة)
- **F** = الرسوم الثابتة
- **VAT** = معدل ضريبة الرسوم (افتراضياً 15%)

### صيغة حساب السعر المقترح

```
S = (C + SH + (F × (1 + VAT)) + TP) / (1 - (P × (1 + VAT)))
```

### حساب الرسوم الإجمالية

```
TotalFees = (S × P + F) × (1 + VAT)
```

### حساب صافي الربح

```
NetProfit = S - C - SH - TotalFees
```

### حساب هامش الربح

```
Margin% = (NetProfit / S) × 100
```

## الميزات المتقدمة

إذا لم تساوِ نسب المتوسط الموزون 100%، يمكنك الضغط على "تسوية تلقائية" لتصحيحها تلقائياً.
